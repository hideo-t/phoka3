## 5. ブランド命名思想

### 5.1 リブランディング戦略

| 旧 → 新 | 意図 |
|---|---|
| 11m → フルサイズ | サイズ訴求から**価値訴求**へ |
| C → グローバルエディション | 内輪用語から**ブランド名**へ |
| 車検対応 + シリーズ → 車検対応型 + トレーラーハウス | 「型」で**製品タイプ明確化** |
| オーダーメイド → フルカスタム | 漢字・和語からカタカナで**訴求力 up** |

**ポイント**: 「シリーズ」を商品名から外し、**それぞれが独立ブランドとして立つ**ように設計。

### 5.2 命名ガイドライン

新製品を追加する場合の命名指針:

1. **カタカナ優先**（フルカスタム / グローバルエディション）
2. **機能を明示**（型 / モデル）
3. **訴求軸を含む**（フル = 全て、グローバル = 国際感、車検対応 = 法規対応）
4. **「シリーズ」「セット」等の内輪用語を避ける**
5. **3-12 文字程度**で短く

### 5.3 NG ワード（命名で避ける）

- `〜シリーズ` / `〜ライン` / `〜セット`
- 型番のみ（C, A, 11m など意味のない記号）
- 二重否定や英字略号（オーダーメイドカスタムオプション等）

## 6. CSS クラス名マッピング

新規 CSS で商品別スタイルを当てる場合の推奨命名（既存スタイル引き継ぎ前提）:

```css
/* 旧クラス名 → 新クラス名マッピング */
.series-11m         → .full-size-trailer    or  .product-fst
.series-c           → .global-edition       or  .product-ge
.series-syaken      → .syaken-type          or  .product-st
.order-made         → .full-custom          or  .product-fc

/* 共通スタイル */
.lineup-card        ← 商品カード共通
.lineup-card-header ← カードヘッダ
.lineup-models      ← モデル名タグ群
```
